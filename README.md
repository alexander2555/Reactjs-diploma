# ResultSchool ReactJS Diploma Project

## Web Collage App

---

**Роли:**

- _Админ_: создание/удаление/редактирование пользователей, документов, назначение редакторов, публикация;
- _Мастер (ЦА)_: создание/редактирование документов, назначение редакторов собственных документов, публикация документов (public/private), загрузка/редактирование графики;
- _Пользователь (Клиент)_: редактирование документов с правом на редактирование, просмотр публичных документов, загрузка/редактирование графики;
- _Гость_: просмотр публичных документов.

---

**Таблицы БД**

| Название      | Поля                                                  | Описание                      |
| ------------- | ----------------------------------------------------- | ----------------------------- |
| **users**     | id, login, password, reg_at, role_id                  | Пользователи                  |
| **roles**     | id, name                                              | Роли пользователей            |
| **documents** | id, title, created_at, owner_id, reader_id, published | Документы (коллажи)           |
| **elements**  | id, title, url, props, uploaded_at                    | Элементы (графика)            |
| **doc_el**    | id, doc_id, el_id, props                              | Связь документов с элементами |
